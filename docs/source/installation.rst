.. _installation:

Installation
============
To install ``spgl1`` within your current environment, type:

.. code-block:: bash

   >> make install

or as a developer:

.. code-block:: bash

   >> make dev-install

To install ``spgl1`` in a new conda environment, type:

.. code-block:: bash

   >> make install_conda

or as a developer:

.. code-block:: bash

   >> make dev-install_conda
